Es bereitet mir viel Freude Neues zu erlernen. Bis jetzt lag der Schwerpunkt meines Studiums auf folgenden Technologien:

Blog in PHP

Diesen Blog habe ich in PHP erstellt. Die Daten werden von einer MariaDB abgerufen und im Dashboard kann man sich einloggen und Einträge verändern.

Chat

Diesen Chat habe ich mit Hilfe von socket.io und Node.js erstellt.
Es werden zwei Chat-Fenster angezeigt, um die Funktionalität nachzuvollziehen.

Coronadaten

Die folgende App ruft täglich mit Hilfe der API des Intensivregisters die aktuelle Auslastung der Intensivstationen durch Coronapatienten ab und speichert die Daten in eine eigene Datenbank. Beim Öffnen der App werden dann alle in der Datenbank gespeicherten Informationen abgerufen und durch ein Diagramm grafisch dargestellt.

Speisekarte

Diese Demo-Speisekarte wurde im Grid-Layout gestaltet und kann mit Hilfe von JavaScript gefiltert werden.
